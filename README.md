# 331-fuzzer

## Setup

Project requires Python 2.7 and pip installed.

First, run

```bash
$ pip install -r deps.txt
```

to install all the required dependencies for this app.

## Example Run

```bash
$ python fuzz.py discover http://127.0.0.1/dvwa/ common_words.txt --auth dvwa
```
